Projeto publicado

SportForYou

Plataforma web desenvolvida para conectar praticantes de esportes, amadores e profissionais, com organizadores de eventos em um ambiente único, acessível e focado no universo esportivo.

PHP JavaScript MySQL
Voltar ao portfólio Ver no GitHub Ver documentação
Tipo Rede social web com foco no universo esportivo
Foco Conexão entre atletas, descoberta de eventos e formação de comunidade
Stack PHP, JavaScript, CSS, MySQL e PHPMailer
Meu papel Desenvolvimento back-end dentro de um projeto em equipe

Problema atacado

O projeto nasceu para resolver uma dificuldade comum: encontrar parceiros, equipes, eventos e espaços para praticar esportes sem depender de várias plataformas desconectadas.

Objetivo da solução

Centralizar em uma única plataforma a interação entre praticantes e organizadores, facilitando comunicação, descoberta de oportunidades esportivas e compartilhamento de experiências.

Principais funcionalidades

  • Feed com postagens, curtidas, comentários e stories
  • Sistema de seguidores e perfis de usuário
  • Mensagens privadas entre usuários
  • Cadastro, interesse e acompanhamento de eventos esportivos
  • Busca de usuários e painel administrativo

Destaque técnico

A estrutura do projeto mostra integração entre front-end e back-end, com organização em páginas, actions, APIs, DAOs e persistência em banco de dados para suportar autenticação, interações sociais e gerenciamento de eventos.

Base de projeto

O desenvolvimento foi guiado por documentação completa, com requisitos funcionais e não funcionais, regras de negócio, modelagem de dados, casos de uso e prototipação em Figma.

Feed social

Publicacoes, curtidas, comentários e stories para aumentar engajamento entre usuários.

Eventos

Cadastro, divulgação e demonstração de interesse em eventos com detalhes como esporte, data e local.

Mensagens

Comunicacao privada entre usuários, ampliando o senso de comunidade dentro da plataforma.

Requisitos e estrutura

Projeto pensado com foco em usabilidade, segurança, compatibilidade, manutenção e crescimento futuro.
